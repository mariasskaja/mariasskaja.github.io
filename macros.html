макрос


Макросы - это записанная последовательность команд, которую затем выполняет Excel. У меня есть три макроса: 1)макрос написания, 2)макрос замены текста, 3)макрос замены шрифта.


1)макрос написания:

Sub сайт()

' '

сайт Макрос

' '

Сочетание клавиш: Ctrl+й

'

ActiveCell.FormulaR1C1 = "akademia-excel.ru"

Range("A2").Select

End Sub


Такой макрос может вставить нужное мне слово/фразу (в данном случае название сайта) в пустую строку. Создаем макрос, добавляем сочетание клавиш для быстрого вызова команды, вводим слово, которое будет позже появляться в пустой строке, сохраняем макрос.


2)макрос замены текста:

Sub замена()

' '

замена Макрос

' '

Сочетание клавиш: Ctrl+ц

'

Cells.Replace What:="akademia-excel.ru", Replacement:="school", LookAt:= _ xlPart, SearchOrder:=xlByRows, MatchCase:=False, SearchFormat:=False, _ ReplaceFormat:=False

End Sub


Такой макрос заменяет введенное слово или фразу на другую. Для этого нужно создать сочетание клавиш, дать команду замены одного слвоа на другое, сохранить макрос.


3)макрос замены шрифта

Sub заменашрифта()

' '

заменашрифта Макрос

' '

Сочетание клавиш: Ctrl+у

'

Cells.Select

Range("J3").Activate

With Selection.Font

.Name = "Berlin Sans FB"

.Size = 11

.Strikethrough = False

.Superscript = False

.Subscript = False

.OutlineFont = False

.Shadow = False

.ThemeColor = xlThemeColorLight1

.TintAndShade = 0

.ThemeFont = xlThemeFontNone

End With

Selection.Font.Italic = True

End Sub


Этот макрос заменяет шрифт слова. Для него также нужно создать новый макрос, сочетание клавиш, команду замену шрифта и сохранить.
